
Simulation d’un réseau d’entreprise – Cisco Packet Tracer

Projet réalisé dans le cadre du cours « Simuler votre réseau » (OpenClassrooms).
L’objectif est de concevoir une maquette réseau d’entreprise fonctionnelle, intégrant :

Un plan d’adressage IP structuré,
Des accès filaires et Wi-Fi sécurisés,
Un sous-réseau VPN pour les clients distants,
Une démarche de tests et de dépannage.

 Objectifs du projet

Comprendre le rôle des équipements d’interconnexion (routeurs, switchs).
Configurer les périphériques finaux (PC, imprimantes, serveurs).
Mettre en place un accès Wi-Fi sécurisé (WPA2).
Intégrer un sous-réseau VPN.
Tester la connectivité et résoudre des pannes.
